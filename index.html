Main Sewer Map

Primary map application used by LRWRA personnel for sewer location and information.

Project Renew

Map of LRWRA's ongoing sewer construction capital projects.

SSLRP

Map of applications received for our Sewer Service Line Replacement Program (SSLRP).

FOG

Map of Fat, Oil, & Grease (FOG) producing facilities that are regulated by LRWRA.

Rain Gauges

Application that allows you to view current rain amounts for LRWRA rain gauges.

SSO Query

Query and map sanitary sewer overflows (SSO) by date. The app, at this time, has limited functionality on mobile devices.

Address Validation

Application used to validate an address. Enter an address in the search bar to check entry against PAgis address data.

3D Sewer

This application models the sewer system subsurface in 3D.

Data Download

A site where you can download LRWRA's Geographic Information System (GIS) data.
